Cara membuat Web Dengan Hexo GitHub

2019-01-24

Tahap Semua Full

halo nama saya Ghaly Kanz Razan hari ini saya akan membuat Tutorial cara membuat situs Web dengan Github
dan Hexo.

  1. kalian harus membuat Akun github di GitHub.com.
  • lalu pilih SignUp jika kalian baru ingin login GitHub => lalu tulislah semua keterangan disitu mulai dari nama, email, dan passwordnya => dan klik SignUp.
  • lalu bagi yang sudah Sign pilihlah SignIn => dan Tulislah email/username-mu => lalu klik SignIn.



  1. lalu ketika sudah SignIn kalian akan masuk ke menu home kalian. Ok selanjutnya kalian hanya perlu membuat Repository untuk membuat project Web kita, dengan klik New Repository => lalu isi lah semua itu seperti nama Repository-mu dan Description-nya => dan klik Create Repository.


  1. kita perlu downlaod terlebih dahulu Git dan NodeJS untuk Command Github nantinya.
  • untuk mendownload Git silahkan Download di git-scm.com
  • untuk mendownload NodeJS silahkan Download di nodejs.org
  1. selanjutnya kita akan membuka Command GitHub untuk Hexo pergilah ke Google dan tulislah GitHub Pages Hexo => lalu pilihlah yang paling pertama => kebukalah Commad-command GitHub untuk Hexo. Disitu kalian bisa melihat semua Commad untuk memerintah Hexo, jadi sebenarnya kalian tidak perlu melihat tutorial berikut.


  1. kita perlu Install hexo melalui CMD. Bukalah CMD yang sudah di Install dengan NodeJS dan Git => lalu ketiklah Command ini
    “npm install -g hexo-cli” => tunggulah installan hexo selesai.

  2. ayo kita buat projectnya sekarang => kita harus membuat Folder baru di data folder misalnya saya membuat folder di Data D: dan berfolder Web Project => bukalah CMD yang sudah di Install dengan NodeJS dan Git => dan ketiklah D: untuk masuk ke Data D => lalu ketiklah “CD” dan alamat folder yang di Inginkan. => lalu di CMD ketiklah Command ini “hexo init nama github.io-mu” => saat sudah masuklah dalam folder github.io-mu dengan command “CD” => lalu tulis command ini untuk install “npm install”.

  1. lalu bukalah Situs Repository kalian di google dan refreshlah => akan berubah tampilan seperti di bawah ini, karena tadi sudah kita install.

  1. saat kita membuka folder github.io kita lalu pilihlah source => post dan “hello world”, bukalah “hello world” itu dengan Chrome atau web browser lainnya. Yap itu adalah tampilan web pertama kita. => lalu bukalah itu dengan Atom untuk pengeditan nantinya.

  1. Ok di bawah adalah contoh saya mengubah Tittle dari web saya dan lainya lainya.

  1. selanjutnya kita perlu mengganti theme dengan cara pergi ke hexo.io/themes. disitu kita ada beberapa macam pilihan Theme seperti saya akan memilih Themenya “Alpha Dust” => pilih theme tersebut => lalu akan di bawa ke situs githubnya lalu klik Clone or Download untuk mendownload Theme tersebut.


  1. lalu extract lah Zip tersebut lalu bukalah foldernya => bukalah Redme untuk membaca semua tentang theme tersebut mulai dari penjelasan dan command-command nya.

WARNING!!!: Membaca Redme dari Theme tersebut sangat lah penting karena semua kegunaannya ada di situ dan semua penjelasan atau Command setiap Theme Berbeda-beda, jadi di wajibkan untuk membaca.

  1. dan jika kalian sudah membaca dan sudah tau penginstallan nya bagaimana mari kita ke tahap selanjutnya, buka CMD lalu ketiklah Command installan Theme kalian seperti punya saya itu adalah “git clone https://github.com/klugjo/hexo-theme-alpha-dust themes/alpha-dust” => lalu Folder Theme tersebut pun akan ada pada Folder Themes dan jika kita buka Atom pun juga bisa jika kita ingin ada pengeditan dalam web tersebut.


  1. lalu kita akan membuka Config.yml pada folder Github.io kita dengan Atom => lalu carilah deploy dan Theme => mari kita ubah semua yang disitu, sperti Theme saya ubah dengan nama Folder theme saya “alpha-dust” dan semua yang ada di deploy ubah sesuai di bawah ini.

  1. ok mari kita buat percobaanya dengan membuka CMD => lalu ketik “hexo server” untuk menjalankan project/repository dalam folder GitHub tersebut. => lalu copyalh situs “Localhost” tersebut dan paste di web browser => hasilnya akan seperti yang di harapkan Thema pun berubah.


  1. Selanjutnya kita harus tau bagaimana cara Post, post About dan memasukan photo.

  2. kita buka lagi CMD lalu tulis Command ini “hexo new Page “nama page”” => di buatlah post baru di folder source => Karena saya post About jadi tidak termasuk dalam folder posts tapi di luarnya => lalu Bukalah about page yang di buat tadi dengan Atom Editlah seperti berikut.



untuk post biasa gunakan saja seperti yang di atas tapi new pagesnya di dalam folder Posts dan pengeditan tittle dan layout boleh berbeda.

  1. Untuk ngepost Photo kita hanya perlu mengisi code seperti di bawah ini.

  • yang di dalam Array adalah nama yang kita inginkan
  • yang di dalam tanda kurung adalah lokasinya Photonya
  1. saat kita deploy dan generate dengan cara ini => kita buka CMD dan Tulis Command ini “hexo deploy” => tunggulah sementara, dan di wajibkan jangan lupa dengan username dan password GitHub mus sendiri karena di deploy ini kamu akan di minta untuk menulis balik akun GitHub mu => saat selesai mungkin ada waktunya untuk generate dulu dengan Command ini “hexo generate –watch” untuk generate => lalu untuk membuka web kita harus membuka situs github.io kita tanpa GitHub.com dan tanpa Command hexo server



jadilah Website kita dengan Menggunakan Hexo dan GitHub.
Untuk sementara terimakasih sudah membaca Tutorial saya Mohon Maaf jika ada kesalahan,
